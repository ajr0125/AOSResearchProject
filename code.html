A Novel Radar Detection Prototype based on Constant False Alarm Rate Concepts

Code

Linked above is a Github repository containing code for a variety of different CFAR detection algorithms, as well as some methods for data generation. As I progress with the research project, I will continue to add new algorithms as I code them myself and eventually the final prototype implementing FOD-CFAR and OSVI-CFAR concepts. I am coding algorithms other than my prototype for eventual performance comparison and to become more familiar with developing CFAR detection algorithms in MatLab.
